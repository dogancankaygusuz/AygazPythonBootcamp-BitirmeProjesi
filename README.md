Taş-Kağıt-Makas Projesi

Özellikler

Tek Oyuncu Modu: Bilgisayara karşı oynayabilirsiniz. Sonuç Ekranı: Her oyunun sonucunu ve skorları görüntüleyebilirsiniz.
